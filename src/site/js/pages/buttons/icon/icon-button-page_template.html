# Icon Button ## Demo The following are the all the available configurations for `wc-icon-button` component. You can open the browser's console to see that there are actions triggered using either click, enter or space bar. For example purposes only, you can click on them to toggle the `notification state`. This can also be achieved by adding the `notification` as an attribute to the `wc-icon-button` component tag.
