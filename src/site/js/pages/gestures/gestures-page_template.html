# Gestures ## Demo The following are the supported gestures as of the lastest version. If you want to test this in a non-touch desktop version then you have to open the dev tools and enable force touch events. ``` > Open Dev tools. > Open the console. > See the three-dot menu in the top left of the console. > Hit Sensors. > Scroll down and force touch events. ```
