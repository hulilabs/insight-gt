# App Bar ## Demo
#### App bar with only the navigation action: Code: ``` html <wc-app-bar v-bind:title="title" titleClass="titleClass" style="margin-bottom: 10px" v-bind:active="true"> <button slot="navigation" style="width: 48px;height: 48px;">X</button> </wc-app-bar> ``` ### App bar with left actions: Notice the gradient in the title when it overflows Code: ``` html <wc-app-bar title="new title new title new title" v-bind:active="true" style="margin-bottom: 10px"> <button slot="navigation" style="width: 48px;height: 48px;">X</button> <button slot="actions" style="width: 48px;height: 48px;">A</button> <button slot="actions" style="width: 48px;height: 48px;">B</button> </wc-app-bar> ``` ### App bar with a link action: SAVE Code: ``` html <wc-app-bar title="new title" v-bind:active="true" style="margin-bottom: 10px"> <button slot="navigation" style="width: 48px;height: 48px;">X</button> <a slot="actions" style="margin: 0 16px 0 16px">SAVE</a> </wc-app-bar> ``` ### App bar without shadow and without navigation: Code: ``` html <wc-app-bar title="This app bar has a compact layout" v-bind:active="true" compact> </wc-app-bar> ``` ### App bar with compact layout (has less height than normal) Code: ``` html <wc-app-bar title="I am almost invisible" v-bind:active="true" style="margin-bottom: 10px" no-shadow> </wc-app-bar> ```
